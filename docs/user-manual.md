### Introduction

The Educational Resource Distribution System helps track and manage the distribution of educational materials such as books, laptops, and other learning resources to schools. It aligns with SDG 4: Quality Education by improving resource management and ensuring equitable access to educational materials.

### Getting Started

#### System Requirements
- Modern web browser (Chrome, Firefox, Edge, Safari)
- Internet connection

#### Accessing the System
1. Open your web browser
2. Navigate to the application URL: `http://localhost:3000`
3. The system does not currently require login credentials

### Dashboard

The Dashboard provides an overview of the system with key statistics and visualizations:

- Total number of schools, resources, suppliers, and distributions
- Resource distribution chart showing quantities by resource type
- Distribution map showing resource allocation by location

### Managing Schools

#### Viewing Schools
1. Click on "Schools" in the sidebar menu
2. The system displays a list of all registered schools

#### Adding a New School
1. Click on "Add New School" button
2. Fill in the required information:
   - School Name
   - Location
   - Contact Person
3. Click "Save" to add the school

#### Editing a School
1. Find the school in the list
2. Click the "Edit" button
3. Update the information
4. Click "Save" to update the school

#### Deleting a School
1. Find the school in the list
2. Click the "Delete" button
3. Confirm the deletion

### Troubleshooting

#### Common Issues

**Issue**: Data not loading
**Solution**: Refresh the page or check your internet connection

**Issue**: Unable to add or update records
**Solution**: Ensure all required fields are filled correctly

**Issue**: Charts not displaying
**Solution**: Try using a different browser or clearing your browser cache
